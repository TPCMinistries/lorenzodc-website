- Email going to spam

- **Fix:** Test different subject lines, check deliverability

- CTA isn't clear
- Link placement is bad
- **Fix:** Shorten email, make CTA more prominent

### **If Clicks Are High but No Completions:**

- Questions are confusing
- Technical issues
- **Fix:** Check assessment flow, test it yourself

- Follow-up is too salesy
- No clear value proposition
- Calendar link broken
- **Fix:** Make follow-up more personal, test calendar

### **If Calls Booked but No Shows:**
- Not enough reminder emails
- Wrong time zones
- No real interest
- **Fix:** Send 24h + 1h reminders, confirm timezone
